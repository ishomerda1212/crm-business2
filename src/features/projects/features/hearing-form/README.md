# Hearing Form feature

`projects/features` 配下にヒアリングフォーム関連のコンポーネント・ロジックを移設するためのプレースホルダーディレクトリです。  
以下の構成を想定していますが、必要に応じて追加・調整してください。

```
hearing-form/
  components/      # 再利用可能なUI
  pages/           # ヒアリングフォーム画面
  hooks/           # フォーム固有のロジック
  data/            # 型定義やスタブデータ
```

外部システムからファイルを移動する際は、このディレクトリ以下に配置してください。

## タブ構成

- 検討箇所
- 重視ポイント
- 完成時期
- 予算・ローン
- 見積状況
- 顧客種別
- お客様情報
- 法人情報
- ご住所情報
- ご職業
- ご家族
